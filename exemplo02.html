Exemplo 02 - Templates como scripts inline

Podemos definir os dados do template dentro de uma tag <script> e incluí-la no documento HTML. Para impedir que o browser execute o código do template, devemos mudar o MIME type para algo diferente de text/javascript. Alguns MIME types comuns são text/html, text/template ou text/mustache. O exemplo a seguir é um simples template usando este método. 

        <script id="sample_template" type="text/html">
            <h1>{{name}}</h1>
        </script>

        <script type="text/javascript">
            var view = {
                name: 'Joe',
                occupation: 'Web Developer'
            };
            $(function(){
                var template = $('#sample_template');
                var output = Mustache.render(template, view);
                $('#person').html(output);
            });
        </script>

Você pode incluir quantos templates desejar no documento com IDs diferentes. Quando quiser usar um template, pegue o HTML dentro da tag script, e passe-o como template.

Como você pode ver, templates são armazenados separadamente e usados dinamicamente quando requisitados. Esse método aumenta a possibilidade de reuso. Entretanto, usando scripts inline limita o escopo do tempalte a uma única página. Se tiver múltplas páginas, você terá que redefinir seus templates novamente. Assim, incluindo seus templates como arquivos externos será uma ótima solução - assim como seus arquivos css.
